MS500

MS500 is a multi-track dataset originated from the website The ‘Mixing Secrets’ Free Multitrack Download Library, featuring 500 professional performed and recorded multi-track projects with multi-track level instrument and bleeding annotations. All multi-track audio files are kept in the format of 44.1kHz and 16 bits. Compared to previous datasets, MS500 contains a longer total mixture duration (34.2h), a bigger averaged number of multi-tracks (20±13), a higher percentage of vocal stem (84%) and a more diverse genre distribution among existing multi-track datasets.

Instrument Level Bleeding Annotation

Construction of MS500

Follow the steps in the supporting code repository.

Multi-track Level Instrument and Bleeding annotations

Instrument annotations were first automatically generated based on the naming convention of multi-track files, then were manually revised to make sure each track only contains one of the 70 instrument labels. During this process, bleeding annotation was also carried out by manually listening to each multi-track. Studio headphone Sony MDR 7506 and audio interface Steinberg UR22C were used. Monitoring sound pressure level was set to approximately 75 dBA. The manual annotation was only done by Huicheng Zhang.

Hierarchy annotation for compatibility

Hierarchy for MedleyDB and MS500 can be found here and here

We first group the most fine-grained 70 instrument labels into 14 instruments, which will then be subsequently clustered into vocal, bass, percussion and other stem. Finally, we define accompaniment by grouping bass, percussion and other together, leaving vocal as another source at this level. Note that we choose this hierarchy to make MS500 compatible to current MSS systems. The optimal hierarchy for music is still worth exploring.

Automatic mixing algorithm for stems generation

We developed an automatic mixing algorithm for all stems using the linear mixing strategy. For Backing Vocal instrument stem, we provided a dedicated algorithm to generate more realistic stereo stem. We first calculate the L&R channel correlation to identify the mono and stereo multi-tracks. For mono multi-tracks, we select those with similar energy salience information and automatically pan them to left or right to create stereo submix. These stereo submixes will be mixed with stereo multi-tracks to create final Backing Vocal stem.
